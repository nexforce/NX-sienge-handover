# Language & Voice

Read this file when writing copy that lives in a deliverable: report headings, slide titles, callout text, email templates, chart titles. The brand-name canonicalization rule lives in `SKILL.md` § "Brand name" — full vocabulary and templates below.

## Name canonicalization

The company name is **Nexforce** — one word, capital N, lowercase rest. Always render it this way in body text, headings, captions, file names, email subjects, and chart titles.

| Form | Use? | Notes |
|---|---|---|
| Nexforce | ✅ | Default in any sentence |
| NEXFORCE | only inside the wordmark logo and short stamp/footer marks | Never in prose |
| NexForce | ❌ | No camelCase |
| nexforce | ❌ | Lowercase only acceptable inside email addresses, domains, file paths |
| Nex Force / Nex-Force | ❌ | Never split or hyphenated |

## Vocabulary

Preferred terms — these reinforce Nexforce's positioning as cross-border infrastructure for LATAM software commerce.

| Use | Avoid | Why |
|---|---|---|
| platform / infrastructure | tool, system, software | Positions as foundational, not as a feature |
| customers / clients | users (in commercial contexts) | Commercial register, not consumer |
| operate / run | execute, launch | Less hype, more operational |
| LATAM | Latam, Latin America, LatAm | Internal standard spelling |
| cross-border | international, global | Specific to the product positioning |
| sellers / providers | vendors, suppliers | Aligns with marketplace vocabulary |
| transaction volume | GMV, sales | Clearer for mixed audiences |

## Tone by language

| Language | Default register | Notes |
|---|---|---|
| **English** | Professional, direct, US spelling | "color" not "colour", "organize" not "organise" |
| **Portuguese (BR)** | Formal but approachable; second-person "você" | Avoid "tu", "vós"; never gerundismo ("vou estar fazendo") |
| **Spanish** | Neutral LATAM Spanish | Avoid Iberian "vosotros"; use "ustedes" |

Default to the language of the source request. If a document mixes audiences, write in English. Never machine-translate brand-critical copy — rewrite natively in each language.

## Numbers, dates, units

- **Currency**: `USD 4.2M`, `R$ 12,5 mi`, `MXN 1.8B` (PT-BR uses comma decimal)
- **Percentages**: `+18%` (no space); deltas as `+3pp` for percentage-points
- **Dates**: ISO `2026-04-24` in tables; "April 24, 2026" / "24 de abril de 2026" in prose
- **Big numbers**: thousand separators per locale — `1,200,000` (EN) vs `1.200.000` (PT-BR)
- **Time**: 24h in tables (`14:30`), 12h in prose for English ("2:30 pm")

## Short email template (EN)

```
Subject: [Topic] — [Action or outcome]

Hi [Name],

[One-sentence context.]

[The ask or update, in one short paragraph. Lead with the verb.]

[Optional: 2–3 bullets only if the email contains a list of items.]

Next step: [single, dated, owned action].

— [Your name]
Nexforce
```

Keep emails under 150 words unless the situation requires more. Subject lines lead with the topic, not with "Re:" or "Fwd:" — strip those when starting a new thread.
